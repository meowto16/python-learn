# 14. List Indexing and Slicing

Доступ по отрицательному индексу:

```python
list = [1,2,3,4,5,6]
list[0] # 1
list[-1] # 6
list[2] # 5
```

Можно забирать только часть листа, с помощью slice

```python
list[0:2] # [1,2,3]
```

Можно реверснуть лист следующим образом
```python
list[::-1] # [6,5,4,3,2,1]
```
